# Masuri
스마트폰 출장수리 서비스 MASURI
>Masuri는 스마트폰 출장 수리, 방문수리등을 예약 할수있는 웹서비스입니다.
## 기능목록
1. 일반회원 페이지

   회원가입 후 수리를 예약할수 있고 공지사항 열람, QNA 게시글 작성등이 가능합니다.

2. 관리자 전용 페이지

   관리자 페이지 회원관리, 직원관리 , 게시글 관리등을 할 수 있습니다.

3. 수리기사 (직원)  전용 페이지

   수리기사의 일정등을 열람하고 관리 할 수 있습니다.

## 개발정보
- 언어 : JAVA 1.8 , JavaScript(ES5) , HTML5 , CSS3 , Oracle SQL , JSP 
- WAS : Tomcat 9.0.38
- DBMS : Oracle XE 112
- FrameWork : Bootstrap 4.3.1
- API : 카카오지도 , 오픈웨더 날씨 , 도로명주소
- Lib : COS , JSTL , MAIL
